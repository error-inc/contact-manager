# **Contact Manager V0.0**

### This is a simple *Command Line Based* **Contact Manager** based in **C**.

## Usage:
### Linux:
>in folder containing manager file  
'./manager contacts.csv'

### Windows:
>in folder containing manager_64bit.exe or manager_32bit.exe file  
'manager_64bit.exe contacts.csv'  
'manager_32bit.exe contacts.csv'

## ***UI***
'  
---------------
Contact Manager
---------------
1.Add Contact  
2.View Contacts  
3.Exit  
Enter choice:  
'
